# connect-africa

This repo contains a lightweight **design system** (tokens + components) inspired by the provided UI screenshots, plus a **single mockup page** that displays all assets.

## Run locally

```bash
npm install
npm run dev
```

Then open the printed local URL (Vite default is `http://localhost:5173`).

## What to look at

- `src/styles/tokens.css`: design tokens (colors, type scale, spacing, radii, shadows)
- `src/styles/components.css`: component styling
- `src/components/*`: reusable components (Button, Input, Card, Navbar, etc.)
- `src/pages/DesignSystemPage.tsx`: mockup page that showcases **all tokens + components**
